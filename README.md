## ğŸ’¡ Project Philosophy
Instead of using heavy JavaScript frameworks for static content, I chose a minimalist approach:

*   **Performance:** With zero render-blocking scripts, this site achieves a high Lighthouse performance score.
*   **Simplicity:** Clean code that is easy to maintain with no vulnerable dependencies (no `node_modules` hell).
*   **SEO Focused:** Equipped with `sitemap.xml` and semantic tags to ensure better search engine indexing.
